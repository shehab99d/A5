Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Returns HTML content (tags included).
innerText: Returns visible text content (no HTML tags, considers visibility).
textContent: Returns all text content (no HTML tags, includes hidden elements).

Question-3: What is event delegation in the DOM?

Less code: Only one event listener is needed, making the code more concise.
Better performance: Especially with a large number of elements, as you avoid attaching multiple listeners.
Flexibility: Works with dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling allows an event to propagate from the target element to its ancestors in the DOM. This behavior can be used for event delegation, making it easier to manage events on dynamically added elements. You can stop event bubbling by calling event.stopPropagation().

Question-5: How do you create, add, and remove elements using JavaScript?

Creating elements: Use document.createElement('elementType').
Adding elements: Use appendChild(), insertBefore(), etc.
Removing elements: Use removeChild() or remove().
These methods let you dynamically manipulate the DOM to create, add, and remove elements.
